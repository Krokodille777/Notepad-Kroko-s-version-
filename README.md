# Notepad-Kroko-s-version-


A simple notepad application built with PySide6, featuring customizable font settings and basic file operations.

## Features
- Text editing area with support for various font styles and sizes.
- Font customization dialog to change font family, size, boldness, italics, and underline.
- Basic file operations: Open and Save text files.
## Installation
1. Ensure you have Python installed (version 3.7 or higher recommended).
2. Install PySide6 using pip:
   ```
   pip install PySide6
   ```
3. Clone this repository or download the source code.
4. Navigate to the project directory.
5. Run the application:
   ```
   python main.py
   ```

## Usage
- Use the "File" menu to open or save text files.
- Use the "Edit" menu to edit the text or change font settings.
- Customize font settings through the font dialog accessible from the "Edit" menu.
